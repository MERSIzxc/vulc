# Vulcan Grade Tracker - Расширение Chrome

## Установка расширения

1. Открой Chrome и перейди на `chrome://extensions/`
2. Включи "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажми "Загрузить распакованное расширение" (Load unpacked)
4. Выбери папку `chrome_extension`

## Как использовать

1. **Запусти сервер Django:**
   ```bash
   cd vulcan_tracker
   python manage.py runserver
   ```

2. **Зайди на сайт:**
   - Открой http://127.0.0.1:8000
   - Зарегистрируйся или войди
   - Скопируй свой API ключ с дашборда

3. **Настрой расширение:**
   - Нажми на иконку расширения в Chrome
   - Нажми "Настройки"
   - Вставь свой API ключ
   - Нажми "Сохранить"

4. **Сохрани оценки:**
   - Зайди на https://uonetplus.vulcan.net.pl/
   - Войди в свой аккаунт
   - Открой страницу с оценками
   - Нажми на иконку расширения
   - Нажми "Сохранить оценки"

5. **Готово!**
   - Оценки автоматически появятся на твоем сайте
   - Обновляй их когда захочешь просто нажав кнопку в расширении

## Примечание

Иконки для расширения нужно создать самому или использовать любые PNG файлы размером 16x16, 48x48 и 128x128 пикселей. Назови их `icon16.png`, `icon48.png`, `icon128.png` и положи в папку `chrome_extension`.
